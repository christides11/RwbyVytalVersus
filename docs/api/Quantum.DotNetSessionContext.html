Class DotNetSessionContext

Namespace
Quantum
Assembly
Quantum.Simulation.dll

This class implements the Photon.Deterministic.IDeterministicSessionContext interface inside the simulation project. It's used on the custom server to access Quantum simulation code like the resource manager and command serialization. A static resource manager is created during Init() that is shared between multiple server simulation instances.
