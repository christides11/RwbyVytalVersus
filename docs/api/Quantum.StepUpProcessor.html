Methods

AfterMoveStep(KCCContext, KCCProcessorInfo, KCCOverlapInfo)

This callback is invoked after each KCC move step - this happens if the KCC moves too fast and the continuous collision detection (CCD) algorithm splits translation vector into multiple smaller steps. During CCD movement the KCCData.DeltaTime is scaled proportionally to size of the step. Not called if KCC.IsActive is set to false.
